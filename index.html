What is this?
Define
Natural Chmod Tool is a free utility to calculate the numeric (octal) or symbolic value for a set of file or folder permissions on Linux servers.
Utility
Toggle the desired switches, or directly enter a valid numeric value (e.g. 755) or symbolic notation (e.g. rwxr-xr-x) to see its value in the other formats.
Explanation
File permissions in the Linux file system are managed in three distinct user classes: owner, group and others/public. Each class can have read, write and execute permissions. A file permission can be represented in symbolic or numeric (octal) format.
